1.2 Quantumult X 共享版的核心特点

  • 多协议支持：兼容 Shadowsocks、V2Ray、Trojan、HTTP 代理等，适应不同的网络需求。

2.2 获取 Quantumult X 共享版配置文件

Quantumult X 本身是一个规则自定义代理工具，用户可以通过 共享配置文件 使其功能更强大。

三、Quantumult X 共享版的配置方法

3.1 导入 Quantumult X 共享配置

Quantumult X 支持两种方式导入共享版配置：

方法 1：使用订阅链接导入

  1. 打开 Quantumult X，进入 “配置” 界面。

  2. 选择“导入”，然后点击**“添加订阅”**。

  3. 输入你的订阅链接（通常是提供商或论坛共享的 HTTPS 网址）。

  4. 确认并保存，Quantumult X 会自动更新规则并导入所有代理节点。

方法 2：手动导入 YAML 配置文件

  1. 在 Quantumult X 里，进入**“设置” → “配置管理”**。

  2. 选择 “导入配置文件”，然后选择本地存储的 YAML 文件。

  3. 应用配置并重启 Quantumult X，新规则即可生效。

3.2 代理服务器设置

Quantumult X 需要代理服务器来工作，用户可以选择：

  • 使用公共免费代理（但可能速度慢，不稳定）；

  • 购买付费代理服务（速度快，稳定性更高）。

设置代理的方法：

  1. 进入 Quantumult X，点击 “代理” 菜单。

  2. 输入代理服务器信息（包括协议类型、服务器地址、端口号、加密方式等）。

  3. 保存并应用设置。

四、如何使用 Quantumult X 共享版？

4.1 连接并启动代理

  1. 打开 Quantumult X，进入主页。

  2. 选择一个代理服务器节点（推荐选择延迟最低的节点）。

  3. 点击“启动”按钮，Quantumult X 开始工作。

  4. 测试连接是否正常（可以打开 YouTube、Netflix 等检查是否能访问）。

4.2 配置分流规则

分流规则可以决定哪些流量走代理，哪些走本地网络，常见设置方式：

  • 全局模式（所有流量都走代理）

  • 智能分流模式（仅代理被屏蔽的网站，其余流量走直连）

  • 手动添加规则（根据自己的需求设置代理流量）

设置方法：

  1. 进入“规则”菜单。

  2. 添加规则（如让 Netflix 走代理，但 YouTube 直连）。

  3. 保存并应用规则。

FAQ

Shadowrocket 如何处理域名直连与代理的冲突？
在 Shadowrocket 的规则管理中优先设置具体规则（如 DOMAIN-KEYWORD 或 DOMAIN-SUFFIX），将直连规则放在规则列表靠前位置，避免被泛用的代理规则覆盖，从而确保特定域名走直连。
WinXray 的节点导入方式有哪些？
WinXray 支持手动添加节点、从 URL 订阅导入节点，以及批量导入 JSON 配置文件。用户可以直接复制订阅地址或粘贴节点链接，程序会自动解析并添加到节点列表中，操作十分便捷。
Clash 节点延迟检测如何帮助分流？
Clash 会定期测试节点延迟，并结合策略组自动选择最优节点处理流量。避免访问高延迟或失效节点，提高网页浏览、视频播放和游戏速度及稳定性，优化整体网络体验。
Hiddify Next 如何设置用户带宽与流量配额？
管理员可为每个用户单独设置带宽和流量上限。超出限制时，可自动降速或阻止访问，确保服务器资源合理分配，避免单用户占用过多带宽，提高整体网络公平性和效率。
Clash 的 Rule Provider 功能有哪些好处？
Rule Provider 可通过远程订阅自动更新规则和节点，减少手动维护。结合策略组，可实现分流规则和节点的自动同步，保证分流效率和网络可用性。
Potatso Lite 如何结合手动节点选择优化访问？
用户可根据网络延迟和用途手动切换节点，选择最优节点处理不同流量。虽然不支持自动更新，但在节点不稳定或特定需求下可以灵活调整，提高访问效率。
VLESS + XTLS 在移动网络的优势？
XTLS 模式减少握手次数和加密开销，适合高延迟或不稳定的移动网络。结合策略组和多节点切换，可提升网页浏览、视频播放和游戏的连接稳定性和流畅度。
Clash 节点测速如何提高访问效率？
Clash 会周期性检测节点延迟和丢包率，根据测试结果自动选择最优节点。结合策略组和分流规则，用户可体验更快的网页加载、视频播放和游戏连接速度，提高整体网络效率。
SSR 如何防止流量被 DPI 检测？
通过协议插件和混淆方式可以模拟正常 HTTPS 或 HTTP 流量，隐藏真实流量特征。合理配置混淆插件和加密方式，可以有效绕过 DPI 检测，保证连接稳定性和长期可用性。
Surfboard 能否同步 iCloud 配置？
可以。Surfboard 支持 iCloud 同步功能，用户可在不同苹果设备上共享同一配置文件。这样即使更换手机或重装系统，也能自动恢复节点与分流设置。
